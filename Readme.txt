======================
 System requirements
======================
  - Windows 7, 8, 8.1, 10
  - Wifi module
  - Internet connection

======================
 How to install
======================
  1) Download "Share Evil.exe"
  5) Run "Share Evil.exe"

======================
 How to use
======================
  1) Enter SSID
  2) Enter password
  3) Select your internet connection
  4) Click "Start Sharing Evil"
